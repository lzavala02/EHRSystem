## 2. Unified Chronic Disease Dashboard
**User Story**: As a **Patient with multiple diagnoses**, I want a single view of my health data from both my primary care provider and my specialized clinics (e.g., Dermatology) so that I don't have to manage my healthcare in isolation.

### Acceptance Criteria
* The dashboard must aggregate data from at least two different network sources (e.g., Epic and NextGen).
* The interface must display a consolidated list of all healthcare providers involved in the user's care.
* Users must be able to view their complete medical history, including height, weight, vaccinations, and family history, even if records are pulled from different systems.
* The system must highlight "Missing Data" fields to prompt the patient to update incomplete records.
