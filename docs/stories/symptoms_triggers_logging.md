## 3. Chronic Symptom & Trigger Logging
**User Story**: As a **Patient managing Psoriasis**, I want to log my specific symptoms, flare-up triggers, and over-the-counter (OTC) treatments within the app so that I can replace my manual notes and provide better data to my doctors.

### Acceptance Criteria
* The system must provide specific fields for "Symptom Description" (e.g., scales, redness, joint aches).
* The logger must include a "Triggers" checklist (e.g., Stress, Lack of Sleep, Scented Products).
* The application must allow users to record non-prescription treatments like multivitamins and specific skincare brands (e.g., Aveeno).
* The system must generate a "Symptom Trend Report" that can be shared with providers via the application.
